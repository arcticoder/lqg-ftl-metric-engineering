## File Naming Convention

All timestamped files follow the pattern: `[component]_[analysis_type]_YYYYMMDD_HHMMSS.json`

- **YYYY**: Year (2025)
- **MM**: Month (07 = July)
- **DD**: Day
- **HH**: Hour (24-hour format)
- **MM**: Minute  
- **SS**: Second
